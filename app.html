User Guide

How to Use the Dashboard

Follow these steps to explore crop health, soil conditions, rainfall, and temperature trends using the Shamba Crop Analysis System on Google Earth Engine.

Step 1: Set the Date Range

In the left sidebar, enter your Start Date and End Date (format: YYYY-MM-DD). The default range spans all three farm seasons: 2025-11-01 → 2026-06-30.

Step 2: Quick Season Select

Use the coloured quick-select buttons to instantly fill the date range for a specific season:

🔘 All Seasons — Nov 2025 – Jun 2026
🟧 Preparation — Nov 2025 – Dec 2025
🟩 Cultivation — Jan 2026 – Mar 2026
🟨 Harvesting — Apr 2026 – Jun 2026

Step 3: Run Full Crop Analysis

Click the "🔍 Run Full Crop Analysis" green button. The system will process Sentinel-2, CHIRPS, and MODIS data for your selected period — allow 30–90 seconds for composites to render. Watch the status message at the bottom of the panel for progress updates.

Step 4: Explore Map Layers

Seven indicator layers and three True Color composites (one per season) are added to the map. Toggle each layer on or off using the Layer List (↗ top-right corner of the map) to compare indicators side-by-side.

Step 5: Switch the Active Layer

Use the "Display Layer on Map" dropdown in the sidebar to select a layer, then click "🔄 Update Map Layer". Options include: NDVI, MNDWI, Soil Moisture, Soil Nutrient, Stressed Vegetation, Rainfall, LST, and True Color per season.

Step 6: Read the Monthly Trend Charts

Scroll down the sidebar to view 8 monthly trend charts — one overview combo chart plus one dedicated chart per indicator. Each covers all 8 months from Nov 2025 to Jun 2026, with linear trendlines included.
